# Overview

As a software developer, I am continually working to strengthen my ability to build interactive, user-friendly applications using modern frameworks. This project focuses on developing a mobile application using Jetpack Compose to better understand state management, UI design, and navigation within Android development.

This app is a simple shopping cart application that allows users to browse a list of products, add items to their cart, and adjust quantities using interactive controls. The app includes multiple screens, navigation between views, and a dynamic cart system that updates in real time based on user input.

The purpose of creating this app was to gain hands-on experience with Android development using Kotlin and Jetpack Compose, as well as to better understand how to manage UI state and user interactions in a modern mobile application.



# Development Environment

The application was developed using Android Studio with Jetpack Compose for building the user interface. Git and GitHub were used for version control and project management.

The app is written in Kotlin and Java and utilizes Android Jetpack libraries, including:
- Jetpack Compose for UI design
- Navigation Compose for multi-screen navigation
- Material 3 components for styling and layout
